How it works

1. You write a post here and click Publish.
2. Admin panel saves the post to posts.json in your GitHub repo via the GitHub API.
3. Your live site at Abhikakadiya2301.github.io immediately reads the updated posts.json and shows the new post.
4. No server, no backend, no database — just a JSON file on GitHub acting as your CMS.

Requires: GitHub token with repo scope saved in Settings above.
